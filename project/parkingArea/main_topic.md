# 주차장 대시보드

## 🚗 스마트 주차장 관리 시스템

### 📊 실시간 주차 현황
- **총 주차구역**: 32개 (기본 격자) → 2개 (고급 분석)
- **빈 자리**: 실시간 감지
- **점유된 자리**: 차량 인식 기반
- **점유율**: 0% (현재 빈 주차장)

### 🔍 감지 기술
1. **YOLO-OBB 기반 차량 감지**
   - CUDA GPU 가속
   - 실시간 객체 인식
   - 높은 정확도

2. **고급 컴퓨터 비전**
   - 주차선 자동 감지
   - 색상 기반 영역 분할
   - 형태학적 이미지 처리

3. **다중 알고리즘 융합**
   - Hough 변환 직선 검출
   - DBSCAN 클러스터링
   - 적응형 임계값 처리

### 🎯 성능 지표
- **감지 정확도**: 90%+ 신뢰도
- **처리 속도**: GPU 가속으로 빠른 분석
- **다양한 접근법**: 4가지 분석 방법 제공

### 📁 시스템 구성
```
parkingArea/
├── realworld_parking.py    # 🏆 최고 성능 고급 분석
├── optimized_parking.py    # ⚡ YOLO 최적화 버전
├── main.py                 # 🔧 기본 OpenCV 방법
├── simple_parking.py       # 💡 경량 PIL 방법
├── run_all_analysis.sh     # 🎮 통합 실행 도구
└── 결과 디렉토리들/
```

### 🛠️ 실행 방법
```bash
# UV 환경에서 실행
uv run realworld_parking.py    # 권장 방법
uv run optimized_parking.py    # 표준 방법
./run_all_analysis.sh          # 통합 비교

# 개발 도구
./dev.sh setup                 # 환경 설정
./dev.sh run                   # 분석 실행
./dev.sh clean                 # 정리
```
